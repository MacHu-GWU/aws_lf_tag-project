Install
------------------------------------------------------------------------------

``aws_lf_tag`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install aws_lf_tag

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws_lf_tag
